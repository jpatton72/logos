Optional offline verse narration via Piper TTS, with a small biblical-name pronunciation lexicon.

## What's new

### Listen to verses (opt-in download)
The Bible reader now has a small speaker icon next to every verse's bookmark. Click it to hear the verse read aloud; click again (or press **Esc**) to stop. Playback is fully offline — nothing leaves your machine after the one-time install.

The voice + binary are **not** bundled with the installer. Open **Settings → Audio (Verse Narration) → Install voice** and Aletheia downloads:

- **Piper** (~25 MB) — the TTS engine, by [rhasspy/piper](https://github.com/rhasspy/piper).
- **en_US-amy-medium** (~63 MB) — a clear American-English voice from Piper's voice catalog.

Roughly **88 MB on disk** after install, stored under your Aletheia user-data directory. **Uninstall** removes everything cleanly.

### Pronunciation lexicon
Generic English TTS engines mangle a lot of biblical names — Mephibosheth, Habakkuk, Nebuchadnezzar, Yahweh, etc. Aletheia ships a small bundled lexicon of ~75 respellings (Mephibosheth → "Mefibosheth", Habakkuk → "Habba-kook", and so on) that's applied to the verse text before it reaches Piper. The lexicon lives at [`src-tauri/src/biblical_lexicon.json`](https://github.com/jpatton72/Aletheia/blob/main/src-tauri/src/biblical_lexicon.json) and grows as testers report missed names — open an issue or PR if you spot a mispronunciation.

### Why opt-in?
The default installer stays close to its current ~30 MB footprint. Users who never want narration pay nothing extra, and users who do want it get a clean one-click download right when they need it.

## Install

### Windows
`Aletheia_x64-setup.exe` (always-latest) or the version-pinned variants on the Releases page.

### Linux
- AppImage: `Aletheia_x64.AppImage` (chmod +x, run)
- Debian/Ubuntu: `Aletheia_amd64.deb` (`sudo dpkg -i`)

Both ship with the same fully populated bundled database as before.
