UX-focused release. Same data + features as v0.1.1; refinements to how you move around the app and read what each piece of data actually means.

## Install (Windows)

Download `Aletheia_x64-setup.exe` (always points at the latest stable build) or the version-pinned `Aletheia_0.1.2_x64-setup.exe` below. Double-click, accept the SmartScreen warning ("More info → Run anyway"), and follow the prompts. If you already have v0.1.0/v0.1.1 installed, this upgrades in place — your notes, bookmarks, reading position, and saved API keys carry over.

WebView2 runtime required (preinstalled on Windows 11 and current Windows 10). If launch fails, install from <https://developer.microsoft.com/microsoft-edge/webview2/>.

## What changed

### Notes: now a sidebar, not a modal
Notes used to take over the whole screen. Now it's a 360px right-side column that lives next to the reader (mirroring the AI panel), so you can keep reading the verse you're thinking about while you write.

- Click any note card to open a wide read-only detail view (max 48rem, 90% viewport height) with the full title, body, all tags, and created/updated dates.
- Inline Edit/Delete on each card; Edit on the detail view.
- Create/Edit form stays a centered modal (focus-trapped, click-outside-safe per v0.1.1).
- Search and Export menu carry over.

### Strong's pop-up: clarification labels
First-time users were guessing what `G2424`, `Heb`/`Gk`, and the floating italic words meant. Every item is now explicitly labeled:

- **HEBREW WORD** / **GREEK WORD** caption above the original-language word.
- *Dictionary form:* prefix on the lemma row (lemma is jargon).
- *Strong's # G2424* (with a hover tooltip explaining James Strong's 1890 numbering system).
- *Hebrew* / *Greek* spelled out in full.
- *Transliteration:* and *Pronunciation:* labels in the lexicon section.
- *Source code:* label on the raw morphology code, with a hover tooltip showing how to read it (`V-IAI-3S = Verb, Imperfect Active Indicative, 3rd Sg`).

### Search → Reader: scroll + flash
Clicking a verse in the search results now navigates to the reader, scrolls the verse into the center of the viewport, and applies a 2.2-second amber flash so your eye lands on the right line. Honors the OS-level "reduce motion" preference.
