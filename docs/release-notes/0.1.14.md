Three new pseudepigrapha books in the Non-Canonical section: **1 Enoch**, **Jubilees**, and **2 Enoch**.

## What's new

The reader's Non-Canonical sidebar section now includes three additions alongside the existing 15 KJV apocrypha books:

- **1 Enoch** (Ethiopic Apocalypse of Enoch) — 108 chapters, 1,049 verses. R.H. Charles's 1917 translation.
- **Jubilees** — 50 chapters, 1,305 verses. R.H. Charles's 1913 translation.
- **2 Enoch / Slavonic Enoch** (Book of the Secrets of Enoch) — 68 chapters, 321 verses. W.R. Morfill's 1896 translation, in R.H. Charles's edition.

All three are clearly public domain in the US (pre-1929) and are full-text searchable through the existing FTS index — try searching for "Watchers", "Methuselah", or "Mastema" to land on passages that don't exist in the canonical 66.

These ship under a new translation entry, **Pseudepigrapha (Public Domain)** (abbreviation `PSEUDO`), with per-book translator credit in its description. You don't need to add it to your active translations to read these books — `get_chapter` now falls back to whatever translation has the content if your selected ones don't cover it. Each verse still shows its own translation tag so you know what you're reading.

## Sources

Texts retrieved from [pseudepigrapha.com](https://www.pseudepigrapha.com/), which republishes the Charles and Morfill translations with light HTML formatting. The committed JSONs under `data/pseudepigrapha/` are the canonical source the build pipeline reads — the scraper at [scripts/scrape_pseudepigrapha.py](https://github.com/jpatton72/Aletheia/blob/main/scripts/scrape_pseudepigrapha.py) is run only when we want to re-pull or add another book, so the build doesn't depend on the upstream site staying up. See [ATTRIBUTIONS.md](https://github.com/jpatton72/Aletheia/blob/main/ATTRIBUTIONS.md) for the full citation chain.

## Caveats

- **1 Enoch 91:14 is missing** from the source transcription (Charles's edition has gaps in the manuscript Charles himself flagged). The verse numbering jumps 1–13 → 15–19. This reflects the actual state of the textual transmission.
- **Charles's reordering** for chapters 91–93 (the so-called "Apocalypse of Weeks") is preserved as Charles published it: 91:1–11 → 91:18–19 → 92 → 93 → 91:12–17. We display these in the canonical chapter-and-verse numbers; the historical reordering is faithful to Charles's 1917 edition.
- **Book of Giants** is not included in this release. Every coherent English version of Giants is a copyrighted modern reconstruction of fragmentary Aramaic Qumran scrolls — Stuckenbruck, García Martínez, or Henning. We're tracking this for a follow-up that ships an explanatory entry pointing to the academic editions, rather than scraping someone else's work.

## Upgrade

Standard. New installs include the pseudepigrapha automatically. Existing installs get them on next launch via the same backfill path that handled the v5 schema additions.
